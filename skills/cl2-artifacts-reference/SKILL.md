# ClusterLoader2 Artifacts Reference

## Overview

This skill provides a reference for navigating the directory structure and understanding the specific CI artifacts produced by ClusterLoader2. It helps the agent efficiently locate logs, metrics, and profiles within the test output.

## Guidelines

1. **Artifact Location:** Identify the root directory for ClusterLoader2 outputs.
2. **Log Navigation:** Use standard paths to find component logs.
3. **Metric Extraction:** Locate JSON and Prometheus metrics files.
4. **Profiling Data:** Find `pprof` files for deep-dive analysis.
